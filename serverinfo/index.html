Cestaberous Minecraft Server Information

CS2

CS2 is the second reincarnation of the modded Cestaberous 1.7.10 server. It is an invite only accessable server. The mods used on the server are not a modpack available for public use. The updates are available only for registered server users. If you are a registered server user, you can use the links below to login to the downloads page to get the latest server update.
