Version 5.3.1 Installation

Instructions:

  1. Close all instances of Minecraft
  2. Download The Mods Using the link below
  3. Extract the ZIP file to a folder on your desktop
  4. Press WIN+R to open the Run Prompt
  5. Type %APPDATA% and press ENTER
  6. Naviate to: .minecraft/mods
  7. Delete the ENITRE CONTENTS of the folder
  8. Copy the extracted files from the folder on your desktop to the mods folder (NOTE: Do not copy the folder)
  9. Close the window and reload Minecraft 1.7.10
